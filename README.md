## 🔖 Sobre

O <strong>Ecoleta</strong> é um marketplace de coleta de resíduos.

- O projeto consiste em uma aplicação com o objetivo de coletas de resíduos que não pode ser posto em lixo comum;

- Permite que as empresas de coletas de resíduos cadastrem os pontos de coletas possibilitando a população encontrá-los de acordo com a localização de sua cidade.

Aplicação web construída na trilha <strong>Starter</strong> da <strong>Next Level Week</strong> distribuída pela [Rocketseat](https://rocketseat.com.br/).

## 🤔 Como executar o projeto

- Faça um fork desse repositório;
- Instale as dependências: `yarn install`;
- Incie o projeto: `yarn start`;
- Acesse o navegador: `http://localhost:3000`.
